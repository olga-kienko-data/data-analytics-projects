# Data Analytics Projects

##  About this repository
This repository contains my data analytics projects created during my studies and practice.
The projects focus on data cleaning, analysis, visualization, and extracting business insights.

##  What problems do these projects solve
The code in this repository helps to:
- clean and prepare raw datasets;
- analyze data using Python and SQL;
- calculate key metrics and KPIs;
- identify trends and patterns;
- support data-driven decision making.

##  Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Tableau
- Power BI
- Google Sheets
- A/B Testing & Statistics

##  Repository structure
- `data/` – datasets used for analysis  
- `notebooks/` – Python notebooks with analysis  
- `sql/` – SQL queries  
- `dashboards/` – dashboards and screenshots  

##  Status
The repository is actively being updated with new projects.
